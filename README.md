# NLP Class Notes

Welcome to my **NLP Class Notes** repository! Here you'll find detailed notes and explanations on the concepts and topics covered in my Natural Language Processing (NLP) course. The notes are designed to help you grasp important concepts and techniques used in NLP, supported by Python code examples where necessary.

The HTML file was created using **Quarto**, an open-source tool designed for creating dynamic and interactive documents. Quarto allows me to combine markdown, code, and visualizations seamlessly into one document, making it easier to explain complex NLP concepts with accompanying code snippets, charts, and graphs.

To access the HTML version of the notes, you can visit the link below:

[View the HTML Report](https://maxwellbernard.github.io/nlp_notes)

## Course Topics
The course includes, but is not limited to:
- **Text Preprocessing**: Tokenization, stemming, lemmatization, and more.
- **Text Representation**: Bag of Words (BoW), TF-IDF, Word2Vec, GloVe.
- **Machine Learning for NLP**: Naive Bayes, SVM, and more.
- **Deep Learning Models**: RNNs, LSTMs, Transformers, BERT.
- **Applications**: Sentiment analysis, named entity recognition (NER), machine translation.
